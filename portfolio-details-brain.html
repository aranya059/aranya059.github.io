Brain Tumor Classification Using Convolutional Neural Network

There are 3 main types of brain tumors: 1. Glioma 2. Meningioma 3. Pituitary. This project is about predicting these types of brain tumors by using the deep learning(Convolutional Neural Network) process. Our model can classify among these 3 types of brain tumor with 93.33% accuracy.
